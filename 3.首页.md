帮我实现 MailHub 的 Next.js 14 首页宣传页（/app/page.tsx）。

## 设计理念
产品叫 MailHub，是一个邮件聚合转发系统。
视觉语言取自中国古代驿站文化：油皮公文纸的底色、毛笔书写的字体韵味、
朱批红的点睛用色、驿站台账的信息排布方式。
不是"中国风装饰"——没有祥云花纹、没有龙纹边框、没有水墨泼洒。
是克制的、材质感的、排版驱动的东方简约。

## 设计规范

### 配色
- 页面底色：#F5EDD6（油皮纸黄，整个页面的统一底色，包括 body）
- 深墨色（主文字/边框）：#1C1410（近黑，带暖棕调）
- 次要文字：#6B5A45（中等棕，像陈旧墨迹）
- 朱砂红（唯一强调色）：#C0392B（用于按钮、状态标记、重要数字、hover 高亮）
- 浅朱砂（背景 tint）：#FAE8E6（极浅，用于 badge 背景）
- 纸面分隔色：#DDD0B3（比底色深一级，用于 border、分割线）
- 卡片表面：#FDF6E4（比底色亮一级，白纸感，用于卡片背景）
- 印泥红（装饰，极少用）：#A93226

### 字体
- 标题：Noto Serif SC（Google Fonts，font-weight 700，有毛笔衬线韵味）
- 正文：Noto Sans SC（font-weight 400，清晰可读）
- 数据/代码：JetBrains Mono（字号降一级使用）
- next/font 引入，不要用 @import

### 形状与边框
- 圆角几乎不用，最多 4px（纸张、印章、台账都是方正的）
- 卡片：border 1px solid #DDD0B3，背景 #FDF6E4，无 box-shadow 或极轻（0 2px 8px rgba(28,20,16,0.06)）
- 按钮：方形为主（border-radius 4px），主按钮用朱砂红填充

### 特殊元素
- Logo 印章：用纯 CSS + SVG 做一个正方形红色印章，内有「驿」字或「МH」两字，
  边框双线（外框 + 内框，间距 3px），背景 #C0392B，文字白色，
  尺寸 36×36px，放在 Navbar 左侧，这是全站唯一的装饰性元素
- 分隔线：用 1px solid #DDD0B3，有时在线两端加极小的菱形装饰点（◆ 字符，字号 8px）
- 台账线：页面某些区域用横向的细线（像账本格子）作为背景纹理，
  实现方式：repeating-linear-gradient(transparent, transparent 31px, #DDD0B3 31px, #DDD0B3 32px)
  仅在「工作原理」section 的右侧卡片背景上使用，不要滥用

## 技术要求
- TypeScript + Tailwind CSS
- tailwind.config.ts 定义所有颜色变量（paper, ink, vermilion, brush 等语义命名）
- next/font 引入 Noto Serif SC + Noto Sans SC + JetBrains Mono
- 组件拆分：Navbar、Hero、DataBelt、HowItWorks、Features、CtaFooter、Footer
- 动画：Framer Motion，仅 fadeInUp（opacity 0→1，y 16→0，duration 0.6，easing easeOut）
  用于 section 进入视口，其余无动画
- 印章 Logo 封装为独立组件 SealLogo.tsx
- 响应式：移动端标题降至 44px，三步骤改竖排，Mockup 隐藏
- prefers-reduced-motion 检测，禁用所有动画
- 无粒子、无渐变、无 glow 效果、无任何 AI 产品惯用视觉套路
